# GlowBuddy 静态界面预览

## 项目概述
这是 GlowBuddy 电子宠物应用的静态界面原型，用于展示应用的用户界面设计和交互流程。

## 包含的页面

### 1. 主页 (index.html)
- 宠物互动界面
- 点击宠物 +1金币
- 长按喂食功能
- 心情显示（开心/一般/难过）
- 饱食度条
- 金币显示

### 2. 商店页面 (shop.html)
- 商品展示（食物、玩具、装饰品）
- 价格显示
- 购买功能
- 库存管理
- 金币余额显示

### 3. AI 对话页面 (chat.html)
- 消息聊天界面
- 用户消息和 AI 回复
- 每日对话次数限制
- 发送消息功能

## 设计特点
- 响应式设计，适配移动设备
- 直观的用户界面
- 清晰的信息层级
- 友好的视觉反馈

## 使用说明
1. 打开 `main.html` 查看所有页面的导航
2. 点击导航卡片可预览相应页面
3. 或者直接打开各独立页面查看
